Single page application

SPA advantages

  • Sending JSON is cheaper (backend advantage)
  • No page flicker
  • Faster UI, more interactive
  • UI is just another client

SPA disadvantages

  • Client must have JavaScript enabled
  • Can take very long to load
  • Sudden reload can take you back to the start of the site
  • Search bots can't index site
